# 要件定義書
## システム概要
### ・システム開発の目的
気軽に作成できる旅行計画ツールを作る
### ・現状の課題
- 誰かが行動的でなければ計画が立たないかつ進まない
- 話の流れで案がいくつか流れてしまう
- 旅行に慣れていないと、何を決めるべきか分からない
- SNS返信の許容期間が人によるため、決めごとが手順よく定まらない
- 予定日合わせは人によって細かさが違うため難しい
### ・システム開発に至った背景
友人と旅行計画を立てるとき、LINEでやり取りすると面倒なことが多く大変だったから
### ・達成されたと言える目標
WebサイトとしてiPhoneで開くことができる
